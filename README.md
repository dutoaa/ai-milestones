# AI Development Milestones - Public Viewer

This is the public static viewer for AI development milestones tracked at https://github.com/dutoaa/2025-ai-development-milestones (private repository).

## About

This page provides a lightweight, searchable view of AI milestones collected from major sources:
- OpenAI, DeepMind, Google AI, Anthropic
- HuggingFace, NVIDIA, AMD
- AWS AI, Microsoft AI

The data is updated automatically via GitHub Actions.

## Data Source

The milestones are collected and curated in the private repository and synced here for public access.

Last updated: Auto-generated on each sync

## View the Site

## Technical Details

- Static HTML viewer with vanilla JavaScript
- No build step required
- Data loaded from `milestones.json`
- Filtering by category, status, and search
